## Config

### Fixed header

Inside `doczrc.js` file there is a `themeConfig.header.fixed` option:

```js
//doczrc.js
export default {
  themeConfig: {
    header: {
      fixed: true,
    },
  },
};
```

#### Options

| Name  | Type      | Default | Description                                       |
| ----- | --------- | ------- | ------------------------------------------------- |
| fixed | `boolean` | false   | Specify if the header should be fixed at the top. |

<hr />

### Logo

Inside `doczrc.js` file there is a `themeConfig.logo` option with a possibility to define logos for light and dark theme.

<Hint type="warning">
  The image has to be placed inside the <InlineCode>public</InlineCode> folder
  or the provided option has to be an url address.
</Hint>

```js
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: '/public/logo.svg',
      width: 50,
    },
  },
};
```

Example with an url:

```js
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: 'https://www.my-website.com/logo.svg',
      width: 50,
    },
  },
};
```

Example with a light and dark theme logo option:

```js
//doczrc.js
export default {
  themeConfig: {
    logo: {
      src: {
        light: '/public/logo.svg',
        dark: 'https://www.my-website.com/logo-dark.svg',
      },
      width: 50,
    },
  },
};
```

#### Options

| Name  |           Type           | Default | Description                                 |
| :---- | :----------------------: | :-----: | :------------------------------------------ |
| src   | `string` &#124; `object` |    â€”    | Specify the image source as an path or url. |
| width | `string` &#124; `number` | `75px`  | Specify the image width.                    |

### Icons

Inside `doczrc.js` file there is a `themeConfig.header.icons` option:

```js
//doczrc.js
export default {
  themeConfig: {
    header: {
      icons: 'minimal',
    },
  },
};
```

#### Options

| Name  |              Type              |  Default  | Description                     |
| :---- | :----------------------------: | :-------: | :------------------------------ |
| icons | `'minimal'` &#124; `'default'` | `default` | Specify the header icons style. |
